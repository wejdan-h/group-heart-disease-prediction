This project applies machine learning techniques to predict heart disease using clinical patient data. The project includes data preprocessing, feature scaling, model training, and performance evaluation using different classification algorithms.

## Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)

## Data Preprocessing

- Missing value checking
- Duplicate removal
- Data standardization using StandardScaler
- Train/Test split

## Evaluation

The models were evaluated using:

- Accuracy
- Precision
- Recall
- Confusion Matrix
- Classification Report

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab


## Purpose

This project demonstrates the complete workflow of a machine learning classification task, including data preprocessing, model training, evaluation, and comparison between different algorithms.
